# Rich Elstrom Construction

Modern website for Rich Elstrom Construction, a luxury custom home builder specializing in Oregon coastal construction.

## Company Focus:

Luxury custom home builders specializing in Oregon coastal homes
Emphasis on Pacific Northwest coastal environments
High-end residential construction including new homes, additions, and remodels

## Core Value Propositions:

"Built to last and be passed on for generations"
Detailed excellence and craftsmanship
Green building practices (Green Project of the Year award)
Coastal specialization

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **CMS**: Sanity (planned)
- **Authentication**: NextAuth.js (planned)
- **Hosting**: Vercel

## Getting Started

First, install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
- `npm run format:check` - Check code formatting

## Project Structure

```
/app
  /components     # Reusable UI components
  /lib            # Utility functions and helpers
  /types          # TypeScript type definitions
  layout.tsx      # Root layout
  page.tsx        # Home page
/components       # shadcn/ui components
/lib              # shadcn/ui utilities
/public           # Static assets
```
